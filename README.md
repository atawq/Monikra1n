# ♚ Monikra1n Tool

**Monikra1n Tool** is an all-in-one Windows utility that provides a modern, user-friendly interface for the **checkra1n** jailbreak while offering quick access to **iFrpfile** tools.

![License](https://img.shields.io/badge/License-MIT-blue.svg)
![Version](https://img.shields.io/badge/Version-0.12.4-green.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)

---

# 🚀 Features

- ✅ **Checkra1n USB Creator**
  - Easily write the Checkra1n ISO to your USB drive with a single click.

- ✅ **iFrpfile Integration**
  - Automatically downloads iFrpfile tools.
  - Installs them into the **Program Files** directory.
  - Creates a desktop shortcut automatically.

- ✅ **Modern User Interface**
  - Built with **CustomTkinter** featuring a clean and modern dark mode.

- ✅ **Automatic Administrator Privileges**
  - Detects when administrator access is required and relaunches itself with elevated permissions.

- ✅ **Safe Operation**
  - Locks the interface during critical operations.
  - Displays a real-time progress bar and status updates.

# 🛠 Installation

Make sure **Python 3.10+** is installed on your computer.

## Clone the repository

```bash
git clone https://github.com/Monikra1nTeam/Monikra1n.git
cd Monikra1n
```

## Install dependencies

```bash
pip install customtkinter pillow
```

## Run the application

```bash
python monikra1n.py
```

---

# ⚙️ Usage

## 💿 Create a Checkra1n USB

1. Connect your USB flash drive.
2. Select the drive under **Target USB**.
3. Click **Start**.
4. Wait until the ISO writing process is completed.

> **⚠️ Warning:** All data on the selected USB drive will be permanently erased.

---

## 📥 Install iFrpfile Tools

1. Click **Download iFrpfile**.
2. The application will automatically:
   - Download the required files.
   - Extract the encrypted archive.
   - Install the tools into **Program Files**.
   - Create a desktop shortcut.
